CSS - Hojas de estilo en cascada El css (acrónimo de Cascading style sheets) es un conjunto de reglas que contienen las declaraciones para dar estilo a los elementos de una pagina web. Su principal objetivo es separar la presentación del contenido.

Documento HTML Un documento HTML es un archivo con extensión .html el cual contiene las etiquetas y el texto plano a mostrar el cual es interpretado por los navegadores (Internet Explorer, Firefox, Chrome, etc) como una pagina web. Con un simple editor de texto plano se puede armar el documento.
